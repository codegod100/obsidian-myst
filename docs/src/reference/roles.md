# Role Catalog

All known role names recognized by the plugin.

## Cross-references

| Role | Content | Purpose |
|------|---------|---------|
| `ref` | Section label | Link to a labeled section |
| `doc` | Document path | Link to another document |
| `eq` | Equation label | Reference a labeled equation |
| `numref` | Figure/table label | Reference a numbered figure |
| `label` | Label name | Define a label target |

## Citations

| Role | Content | Purpose |
|------|---------|---------|
| `cite` | Citation key | Full citation |
| `cite:p` | Citation key | Parenthetical citation |
| `cite:t` | Citation key | Textual citation |
| `cite:ps` | Citation key | Parenthetical, short form |
